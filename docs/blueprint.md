# **App Name**: TuneDetective

## Core Features:

- Text Detection from Audio: Detect and extract text-based metadata (artist, title) from audio files in the Kaggle dataset using AI tool.
- Logistic Regression Model Training: Train a logistic regression model on the extracted text metadata from the Kaggle dataset to predict music preferences.
- Music Recommendation: Recommend songs based on the trained logistic regression model, using user input or previously liked songs.
- User Interface: A user interface (UI) where users can search for music and view recommendations.
- Song Playback: Implement a basic audio player to play song previews. Only play the portion of each music, do not allow to store or download full length songs to avoid violation of copyrights.

## Style Guidelines:

- Primary color: Deep Indigo (#4B0082) to evoke a sense of sophistication and musical depth.
- Background color: Light Gray (#D3D3D3) to provide a neutral backdrop that highlights the content.
- Accent color: Electric Purple (#BF00FF) for interactive elements and highlights, complementing the primary color.
- Body and headline font: 'Inter', a sans-serif font that offers a modern, machined, objective, neutral look; suitable for both headlines and body text.
- Use simple, modern icons for music controls and categories.
- Clean and intuitive layout with a focus on easy navigation.
- Subtle transitions and animations when switching between songs and recommendations.
